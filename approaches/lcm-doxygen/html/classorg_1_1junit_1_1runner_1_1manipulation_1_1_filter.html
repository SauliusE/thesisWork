Detailed Description

The canonical case of filtering is when you want to run a single test method in a class. Rather than introduce runner API just for that one case, JUnit provides a general filtering mechanism. If you want to filter the tests to be run, extend Filter and apply an instance of your filter to the org.junit.runner.Request before running it (see org.junit.runner.JUnitCore#run(Request). Alternatively, apply a Filter to a org.junit.runner.Runner before running tests (for example, in conjunction with org.junit.runner.RunWith.

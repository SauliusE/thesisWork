Detailed Description

A test for the Chart2D that constantly adds new tracepoints to a Trace2DLtd. Mainly the runtime- scaling is interesting.

Furthermore this is an example on how to connect other components to the Chart2D using an adaptor- class. If interested have a look on info.monitorenter.reflection.ObjRecorder2Trace2DAdapter.

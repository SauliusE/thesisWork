void info.monitorenter.util.collections.TestTreeSetGreedy.testMultiThreadingAddRemove ( )
inline

Starting 20 Threads that will remove and add inital equal elements (with an internal comparableProperty of ITrace2D#ZINDEX_MAX) each 50 times with arbitrary sleep times. Each Thread will ensure that it's own IComparableProperty will be removed from the TreeSetGreedy after removing it and assert this by calling the remove call a 2nd time and looking for the returned boolean. The main Thread will wait until all Threads have terminated.

Helper class that iteratively adds an element, asserts that the operation was successful and then removes it again and asserts again that removing was successful.

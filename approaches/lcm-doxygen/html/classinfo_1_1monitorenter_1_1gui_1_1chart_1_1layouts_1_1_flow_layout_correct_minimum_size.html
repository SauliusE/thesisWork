Detailed Description

A flow layout that claims the correct height of the component managed in case the available width is known. The standard java.awt.FlowLayout does not claim the correct size but chooses the maximum width of all components to render which is worthless as the fact of flow breaks is not taken into account.

This class is inspired by the sun class java.awt.FlowLayout with modifications to the methods preferredLayoutSize(Container) and minimumLayoutSize(Container).

Detailed Description

A Failure holds a description of the failed test and the exception that was thrown while running it. In most cases the org.junit.runner.Description will be of a single test. However, if problems are encountered while constructing the test (for example, if a org.junit.BeforeClass method is not static), it may describe something other than a single test.

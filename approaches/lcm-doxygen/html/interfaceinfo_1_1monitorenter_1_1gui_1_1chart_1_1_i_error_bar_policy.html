Detailed Description

Interface for a facade towards painting error bars that adds a layer of configuration.

It acts as a facade for an info.monitorenter.gui.chart.ITracePainter with configurable info.monitorenter.gui.chart.IErrorBarPainter instances that will be provided with configured info.monitorenter.gui.chart.IErrorBarPixel instances.

final String info.monitorenter.gui.chart.IErrorBarPolicy<>.PROPERTY_CONFIGURATION = "IErrorBarPolicy.PROPERTY_CONFIGURATION"
static

The property key defining a general change of an instance.

This is fired whenever the internal configuration of the error bar policy changes. This internal configuration should be of no interest for clients of this interface. What counts is that rendering of the error bars will have changed whenever this event is fired. Subclasses might fire this event to tell exactly this: "Rendering has changed. Please repaint."

As it is of no interest and knowledge which configuration has changed the java.beans.PropertyChangeEvent#getNewValue() and the java.beans.PropertyChangeEvent#getOldValue() of the java.beans.PropertyChangeEvent given to PropertyChangeListener#propertyChange(java.beans.PropertyChangeEvent) of listeners should be null always.

Use in combination with addPropertyChangeListener(String, PropertyChangeListener).

Detailed Description

A general purpose PopupListener.

It is used to connect JPopupMenu instances with the components retrieved from factory methods (of factory info.monitorenter.gui.chart.controls.LayoutFactory).

Note that instances have to be registered as a listener on components via java.awt.Component#addMouseListener(java.awt.event.MouseListener) to make it working.

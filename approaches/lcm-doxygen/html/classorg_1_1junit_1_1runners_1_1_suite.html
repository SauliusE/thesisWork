Detailed Description

Using Suite as a runner allows you to manually build a suite containing tests from many classes. It is the JUnit 4 equivalent of the JUnit 3.8.x static junit.framework.Test suite() method. To use it, annotate a class with (Suite.class) and ({TestClass1.class, ...}). When you run this class, it will run all the tests in all the suite classes.

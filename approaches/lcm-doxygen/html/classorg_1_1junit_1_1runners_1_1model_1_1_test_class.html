org.junit.runners.model.TestClass.TestClass ( Class<?>  klass)

Creates a

TestClass

wrapping

klass

. Each time this constructor executes, the class is scanned for annotations, which can be an expensive process (we hope in future JDK's it will not be.) Therefore, try to share instances of

TestClass

where possible.
